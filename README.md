# 🌐 Nashrah Portfolio Suite

A full-stack portfolio ecosystem showcasing modern application development using **ASP.NET Core**, **.NET MAUI**, **Next.js**, and **SQL Server**.

---

## 🚀 Overview

This repository contains the backend, frontend, and mobile components of my portfolio platform.

The solution demonstrates:

* Clean Architecture
* Cross-platform development with .NET MAUI
* RESTful API development with ASP.NET Core
* Modern web development with Next.js and TypeScript
* Scalable project organization
* MVVM design pattern
